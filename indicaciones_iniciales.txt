Contexto General
Debo realizar una adaptación completa de la aplicación "Synapsis", actualmente orientada a un servicio de Neurocirugía del Hospital de Niños de La Plata. El nuevo enfoque está dirigido a médicos pediatras que trabajan en consultorios particulares, atendiendo pacientes desde recién nacidos hasta los 15 años, con énfasis en bebés y niños de corta edad.

Objetivos Principales
    1. Eliminar funcionalidades obsoletas para el nuevo contexto.
    2. Adaptar funcionalidades existentes al nuevo flujo de trabajo pediátrico.
    3. Agregar nuevas funcionalidades críticas para el consultorio pediátrico.
    4. Limpiar el repositorio eliminando archivos no esenciales para esta nueva versión.

Eliminación de Módulos (Quitar solapas)
    • "Cirugías" → No aplica a pediatría ambulatoria.
    • "Pase" (pase de guardia) → No aplica a consultorio particular.
    • "Stock" → No aplica a consultorio particular.

Nuevas Funcionalidades
1. Base de Datos de Pacientes + Comandos de Voz (CRÍTICO)
    • Reemplazar las fichas físicas de papel por un sistema digital.
    • Comando de voz: Permitir cargar datos de pacientes (nombre, diagnóstico, etc.) usando el micrófono nativo del celular.
    • Pregunta para análisis técnico:
        ◦ ¿Es factible implementar esto sin costos excesivos de tokens/API? (ej. usando reconocimiento de voz offline o nativo del dispositivo).
        ◦ ¿Qué librerías o herramientas recomiendas para iOS/Android?
        ◦ ¿Cómo se integraría con la base de datos existente?
2. Generación de Historia Clínica / Informe Pediátrico
    • Opción para generar un informe pediátrico en PDF o documento editable.
    • Debe incluir:
        ◦ Datos personales del paciente.
        ◦ Antecedentes.
        ◦ Controles realizados.
        ◦ Vacunación.
        ◦ Diagnósticos y tratamientos.
    • Debe ser descargable o imprimible desde el consultorio.
3. Funcionalidad Opcional (a considerar)
    • "Cuadernito de Crecimiento Digital":
        ◦ Reemplazar el cuadernillo físico que llevan las madres.
        ◦ Registrar talla, peso, perímetro cefálico.
        ◦ Mostrar evolución automática según curvas de crecimiento (OMS).
        ◦ Generar alertas si el percentil está fuera de rango.
        ◦ Enfoque principal en niños de 0 a 2 años.
